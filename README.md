# TrustACG Website & Learning Ecosystem

Welcome to the **TrustACG** digital platform, a comprehensive learning and consultancy ecosystem designed to empower professionals in Cybersecurity, AI, Fintech, and more.

## 🚀 Core Objectives

The TrustACG platform is built to support three primary pillars:

1.  **Training**: A robust Learning Management System (LMS) integrated with Thinkific, offering a wide range of specialized courses.
2.  **Consultancy**: A professional showcase of our services with integrated lead generation for business growth.
3.  **Resources**: A rich library of articles, whitepapers, and blogs providing deep industry insights.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **UI Components**: [React 19](https://react.dev/), [Phosphor Icons](https://phosphoricons.com/), [Lucide React](https://lucide.dev/)
- **Animations**: [Embla Carousel](https://www.embla-carousel.com/), [tw-animate-css](https://github.com/Yushivki/tw-animate-css)

## 🌐 Platform Architecture

### Subdomain Strategy
- **Primary Domain (`trustacg.com`)**: Landing page, Resources, Consultations, and About Us sections.
- **Training Subdomain (`training.trustacg.com`)**: Course catalog and full Thinkific LMS integration.

### Page Hierarchy
- **Landing Page**: A high-impact hub featuring hero sections, course overviews, and consultancy summaries.
- **Resources**: A dedicated CMS-driven library for blogs and whitepapers.
- **Consultations**: Specialized vertical-specific pages (GRC, Fintech, Cyber Privacy, etc.).
- **Auth System**: Integrated registration for both Students and Trainers.

## 📦 Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Trust-Axis-LMS/trust-axis-lms.git
   ```
2. Navigate to the frontend directory:
   ```bash
   cd trust-axis-lms/frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🔗 Integrations
- **LMS (Thinkific)**: Handles all course hosting, progress tracking, and student learning environments.
- **Mailing System**: Automated triggers for enrollments, onboarding approvals, and newsletters.
